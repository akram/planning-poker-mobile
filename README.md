Progressive web app de planning poker feita com o [framework ionic](http://ionicframework.com/).

### Installing

``` bash
$ git clone https://github.com/planning-poker/planning-poker-game.git
$ npm install
```

### How to use it

``` bash
$ npm start
```

### Build

``` bash
# mobile package
$ npm run pkg

# web package
$ npm run webpkg
```
